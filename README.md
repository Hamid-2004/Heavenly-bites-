# Heavenly Bites

Premium artisan bakery landing page built with Next.js 15, TypeScript, Tailwind CSS, and Framer Motion.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Replace Images

Swap placeholder images in `public/images/`:

| File | Used In |
|------|---------|
| `hero-cake.jpg` | Hero section featured cake |
| `cake-1.jpg` – `cake-6.jpg` | Best Sellers product grid |
| `about-1.jpg` – `about-4.jpg` | About section collage |
| `avatar-1.jpg` – `avatar-4.jpg` | Hero customer avatars |

## Customize

- **Brand & contact info:** `lib/constants.ts`
- **Products & features:** `lib/data.ts`
- **Colors & fonts:** `styles/globals.css`

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS v4
- Framer Motion
- Lucide React
